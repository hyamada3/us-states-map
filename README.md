# 🇺🇸 アメリカ合衆国 50州インタラクティブ地図

アメリカ合衆国の50州を地図上に表示し、クリックで州名を確認できるデスクトップ向けWebアプリです。

## 機能

- **50州の地図表示** — D3.js + AlbersUSA投影法（アラスカ・ハワイのインセット付き）
- **地域別カラーリング** — 北東部・南部・中西部・西部を色分け表示
- **クリックで州名表示** — 初期状態は州名なし、クリックで略称をトグル表示
- **サイドバー詳細** — クリックした州の英語名・日本語名・地域を表示
- **ウィンドウリサイズ対応** — レスポンシブレイアウト

## 使い方

### 方法1: HTMLファイルを直接開く
`us_map.html` をダブルクリックしてブラウザで開く

### 方法2: Pythonから起動
```bash
python run_map.py
```

## 必要環境

- モダンブラウザ（Chrome / Edge / Firefox）
- インターネット接続（地図データをCDNから読み込むため）

## 技術スタック

| 技術 | 用途 |
|---|---|
| D3.js v7 | 地図描画・インタラクション |
| TopoJSON | 州境界データ |
| us-atlas v3 | アメリカ州地図データ |
| HTML / CSS / JavaScript | フロントエンド |
| Python (オプション) | ブラウザ起動ランチャー |
